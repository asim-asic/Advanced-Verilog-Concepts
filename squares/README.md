# Squares Using Function (Verilog)

## Description
This project demonstrates the use of a Verilog function to compute the square of a 4-bit input number. The function performs combinational logic, while the result is stored using sequential logic (clocked behavior).

## Key Learning
Functions in Verilog are combinational, execute in zero simulation time, and return a single value. When used with non-blocking assignments inside a clocked block, the output is registered and appears with a one clock cycle delay.

## Concept Used
- Verilog Functions
- Blocking vs Non-blocking assignment
- Sequential logic using clock (posedge clk)
- Function call inside always block
